# ASOUL 直播弹幕分析

这个项目用于整理 ASOUL 直播弹幕数据，并产出一套精简后的核心分析报告。

当前主线分成两层：

- 数据构建层：把 JSON / XML 统一整理成 `all_live_details.csv`，再生成黑名单和用户总表
- 报告层：围绕“kg、510回旋、广义回旋、生日会舰长画像”生成少量核心表和图

## 目录

- `弹幕JSON/`
  正式直播 JSON 原始数据。

- `XML弹幕文件/`
  历史 XML 弹幕回放数据。

- `场次明细表/`
  单场聚合后的明细表。
  - `场次明细表/弹幕JSON/`
  - `场次明细表/XML弹幕文件/`
  - `场次明细表/all_live_details.csv`

- `用户总表/`
  从统一明细进一步汇总得到的用户长表 / 宽表。

- `黑名单/`
  当前黑名单规则：
  `2025-06-09 00:00:00` 至 `2025-12-08 23:59:59` 期间发过言的 UID。

- `reports/`
  核心报告代码。
  - `reports/core.py`：统一记录清洗与过滤逻辑
  - `reports/build_report.py`：生成默认版 / 过滤版核心报告

- `分析结果/核心报告/`
  当前正式输出目录。
  - `分析结果/核心报告/默认版/`
  - `分析结果/核心报告/去0211单次且无后续/`

- `分析结果/legacy/`
  旧版承接分析与旧图表，已归档，不再作为主报告。

## 根目录脚本

- `single_live_to_detail.py`
  读取 `弹幕JSON/` 与 `XML弹幕文件/`，生成单场明细和统一总表。

- `build_uid_blacklist.py`
  基于统一明细重算黑名单。

- `build_user_summary.py`
  基于统一明细生成用户总表。

- `build_all_reports.py`
  一键串联数据构建层与核心报告层。

- `analysis_settings.py`
  统一时间窗口与常量配置。

- `user_tagging.py`
  统一用户标签逻辑。

## 推荐流程

直接运行：

```bash
python3 build_all_reports.py
```

这会依次产出：

- `场次明细表/all_live_details.csv`
- `黑名单/uid_blacklist.csv`
- `用户总表/user_summary_wide_clean.csv`
- `分析结果/核心报告/默认版/`
- `分析结果/核心报告/去0211单次且无后续/`

如果只想单独重跑报告层：

```bash
python3 -m reports.build_report --variant-name 默认版

python3 -m reports.build_report \
  --variant-name 去0211单次且无后续 \
  --exclude-source-single-pass-session-key 32d14305-7ef3-4b90-b261-8d68e754b6b3
```

如果希望第一次重构时顺手把旧版输出归档到 `分析结果/legacy/`：

```bash
python3 -m reports.build_report --variant-name 默认版 --archive-legacy
```

## 当前口径

- 核心直播固定为：`乃琳_鸣潮`
- 分析窗口起点：`2025-12-09 00:00:00`
- `kg`：`2025-12-09` 及之后首次被观测到，且首次互动直播就是 `乃琳_鸣潮`
- 主核心口径：`kg 在首次进入乃琳_鸣潮后，累计在 A-SOUL 任意直播有效到场>=5`
- 敏感度阈值：`>=3 / >=5 / >=7`
- 510回旋：`2022-06-01` 及以前出现过，`2022-06-02` 至 `2025-12-08` 完全沉默，`2025-12-09` 后重新出现
- 广义回旋：`2025-12-09` 前出现过、`2025-12-09` 后再次出现，且不属于 510 回旋
- 默认版：保留全部 `乃琳_鸣潮` 触达用户
- 过滤版：排除 `乃琳_鸣潮_0211` 中“首次命中 `乃琳_鸣潮` 且之后再未出现”的一次性用户

## 这次重构后的重点

旧版报告更偏“来过多少人”。  
新版核心报告只回答四个问题：

- 在精确定义下，kg 里有多少能成为核心观众
- 510 回旋和广义回旋两个集合各有多少
- 承接率到底对应的是哪种用户行为定义
- `生日会/20260314-194426-嘉然今天吃什么-嘉然生日会以爱之名843.xml` 里的舰长分别来自哪里
