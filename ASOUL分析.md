# 基于 43 场直播、287万条弹幕数据的鸣潮时代ASOUL活跃观众承接分析

## 一、分析主题

本文围绕 `乃琳_鸣潮` 人群展开，目标是回答两个问题：

1. 乃琳鸣潮带来的增量人群，在后续被团内其他成员承接的效果。
2. 这批人群的团内流动方向，究竟更偏向嘉然、贝拉，还是继续留在乃琳。

当前结论基于弹幕 / 礼物活跃用户数据，因此更准确地说，是对**活跃观众后续承接**的分析，而不是对全体同接用户的完整追踪。

## 二、数据口径

### 1. 数据来源

项目使用来自`Danmakus站`的直播弹幕 JSON 作为原始数据，感谢`Danmakus站`的数据统计

对于一直在的老东西——项目使用了https://github.com/esterTion/BiliBili_crc2mid对`2025.12.9`之前的官号直播回放(包括25年三位成员的生日会)yml弹幕文件的CRC哈希值反向解析为对应的用户UID总结成为黑名单。

在这里，笔者认为黑名单中`4500+`的样本数具有一定说服力。

(但由于缺失比`2026.1.24`更早的数据，很遗憾没能从乃琳直播鸣潮开始分析，但分析仍在嘉然第一次播鸣潮之前。如果有朋友能提供更早的弹幕UID数据，欢迎联系我)

当前主分析窗口为：

- 起点：`2026-01-24 11:55`
- 终点：`2026-03-05 20:53`

在这个窗口内，稳定的 `乃琳_鸣潮` 用户 规模为：`31609`

其中：

- 窗口内首次观测即出现在  `乃琳_鸣潮` 用户 的占比为 `84.27%`
- 在成为 `乃琳_鸣潮` 用户 之前已在别处出现过的用户占比为 `15.73%`

这说明：当前窗口中的 `乃琳_鸣潮` 人群，仍然在主体上可以视为一批在该窗口内被 `乃琳_鸣潮` 首次捕获的人群，但并不等同于“全历史绝对新粉”。

### 2. 时间口径

本文所有“后续人员流动”均使用 `后续承接` 口径：

- 先定位每个用户**第一次进入 `乃琳_鸣潮`** 的时间
- 只统计该时间之后，他去到的其他目标直播

因此，本文重点讨论的是：

- **乃琳鸣潮之后，这批人往哪里流动**，而不是简单的人群重合。

### 3. 解释边界

本文不能直接等同于“同接转化率”，原因有三点：

1. 数据样本是活跃弹幕 / 礼物用户，不包含沉默观众。
2. 当前缺少更早期的 ASOUL 全量直播数据，因此不能做完整生命周期追踪。
3. 不同直播类型的场次数不同，因此更适合比较方向和结构，而不适合直接把这里的比例机械等同为同接比例。

不过，本文仍然可以较好回答：

- 乃琳鸣潮是否像一个增量入口
- 增量入口形成后，团内谁承接得更多
- 这批人最终是单向流向别人，还是形成多成员观看

## 三、核心结论

### 结论 1：乃琳鸣潮确实构成了显著的增量入口

从窗口内观察结果看，`乃琳_鸣潮` 共吸纳 `31609` 名 source 用户，其中 `84.27%` 的用户在当前窗口内第一次被观测到时就出现在 `乃琳_鸣潮`。

这意味着：

- `乃琳_鸣潮` 不只是一个普通内容类型直播
- 它在当前窗口内承担了**明显的人群入口功能**

结合你对同接的体感观察，这一点是成立的：鸣潮之后，ASOUL 的整体盘子被明显放大，而不是只在乃琳单人直播内部做存量迁移。

### 结论 2：增量人群后续流动的第一方向是嘉然，而不是贝拉

在所有后续去向中，最强的目标是 `嘉然_鸣潮`：

- `post>=1`：`19350` 人，占 source `61.22%`
- `post>=2`：`14200` 人，占 source `44.92%`
- `post>=3`：`10785` 人，占 source `34.12%`
- `first_target`：`14711` 人，占 source `46.54%`

也就是说，将近一半的 source 人群，在进入 `乃琳_鸣潮` 之后，**第一跳就去了嘉然鸣潮**。

如果把嘉然所有类型合并看，后续去过嘉然直播的人群占 source：

- `63.43%`

这是一个非常高的比例，说明乃琳鸣潮带来的增量，并没有只停留在乃琳自己的内容池里，而是明显向嘉然方向外溢。

### 结论 3：贝拉也有承接，但明显弱于嘉然

贝拉方向中最重要的承接点是 `贝拉_终末地`：

- `post>=1`：`5073` 人，占 source `16.05%`
- `post>=2`：`1982` 人，占 source `6.27%`
- `post>=3`：`880` 人，占 source `2.78%`
- `first_target`：`2554` 人，占 source `8.08%`

如果把贝拉所有类型合并看，后续去过贝拉直播的人群占 source：

- `20.19%`

这说明贝拉并不是没有吃到乃琳鸣潮带来的流量，但相对于嘉然，承接强度明显更弱。

因此，如果用一句非常直白的话概括这部分结果，就是：

- **嘉然吃到最多外溢，贝拉吃到最少外溢。**

这与我的体感判断是基本一致的。

### 结论 4：乃琳自己并没有因为外溢而失去承接能力

如果把乃琳 source 之后的其他类型直播合并看，后续去过乃琳其他直播的人群占 source：

- `39.88%`

进一步看解释性阈值结果：

- 乃琳 `post>=1`：`59.08%`
- 乃琳 `post>=2`：`41.36%`
- 乃琳 `post>=3`：`30.50%`
- 乃琳 `post>=5`：`17.37%`

这说明：

- 乃琳鸣潮不是“把人送出去之后自己失血”
- 这批人中仍有大量用户会继续回流到乃琳的其他直播类型

所以更准确的结构不是“乃琳拉新、别人吃走”，而是：

- **乃琳先打开入口，随后一部分人留在乃琳，一部分人流向嘉然，少部分流向贝拉。**

### 结论 5：这不是单向外流，而是典型的团内扩散结构

从三人层面的流动分层看：

- 无后续到场：`30.48%`
- 仅留在乃琳：`4.84%`
- 只去嘉然：`26.00%`
- 只去贝拉：`0.79%`
- 乃琳 + 嘉然：`18.50%`
- 乃琳 + 贝拉：`0.47%`
- 嘉然 + 贝拉：`2.86%`
- 三人都看：`16.07%`

这个结构很有信息量。

首先，`只去嘉然` 明显高于 `只去贝拉`，再一次证明嘉然是最强的跨成员承接方向。

其次，`乃琳+嘉然` 与 `三人都看` 两项合计超过三成，说明相当一部分人不是简单地从 A 跳到 B，而是逐步扩展成**多成员观看**。

因此，乃琳鸣潮带来的并不只是短期单点流量，而更像是一次团内观看结构的放大：

- 一部分人被乃琳留下
- 一部分人被嘉然吸收
- 一部分人被扩散成多成员用户

从团体视角看，这比单人留存本身更重要。

## 四、与体感数据的对应关系

### 1. “嘉然播鸣潮吸到最多潮U”

数据里嘉然方向是最强承接方向：

- 去过嘉然直播：`63.43%`
- 第一跳到嘉然鸣潮：`46.54%`
- 三人阈值中嘉然 `post>=1` 也是最高：`63.43%`

所以“嘉然吃到最多鸣潮外溢”这一点，和我的体感高度一致。

### 2. “贝拉播终末地吸到最少潮U”

数据里贝拉方向明显弱于嘉然：

- 去过贝拉直播：`20.19%`
- 第一跳到贝拉终末地：`8.08%`
- 三人阈值中贝拉 `post>=1` 只有 `20.19%`

因此，“贝拉吸到的鸣潮流量最少”这一点也和结果一致。

### 3. “乃琳自己直播依然很高”

虽然嘉然承接很强，但乃琳后续承接同样不低：

- 去过乃琳其他直播：`39.88%`
- 三人阈值中乃琳 `post>=1`：`59.08%`

所以从结构上看，乃琳既是入口，也是留存位。

### 4. 团播的数据疑问?

当前数据能说明：

## 五、为什么会出现这样的结构

### 1. 内容相近性决定了第一跳优先去嘉然鸣潮

在 source 之后，最强首跳目标是 `嘉然_鸣潮`，这说明这批人并不是随机团内流动，而是先沿着“鸣潮内容”继续移动。

也就是说，鸣潮带来的是一批**内容驱动型入口用户**，他们进入团体后，优先选择同样还在播鸣潮的目标。

这个现象解释了为什么嘉然方向会特别强：

- 同为鸣潮内容
- 嘉然又具备较强的个人吸附能力
- 因此最容易吃到第一波承接

### 2. 贝拉方向的承接更像次级外溢，而不是第一主路径

贝拉最强承接点是 `终末地`，本身也是游戏向内容，因此它确实能接住一部分鸣潮用户。

但无论从 `post>=1`、`post>=2` 还是 `first_target` 看，贝拉都明显低于嘉然。这说明贝拉不是这批鸣潮增量的第一主路径，更像是第二层、第三层的后续分流点。

### 3. 多成员观看的形成，是团体盘放大的关键信号

`乃琳+嘉然`、`嘉然+贝拉`、`三人都看` 合计规模并不小，这意味着乃琳鸣潮带来的并非单纯的“单点吸粉”，而是带来了一批更可能发展成团体型观看的潮u。

从团体经营角度看，这类用户价值往往高于一次性内容用户，因为他们未来更容易继续流向团播、周年、3D、突击等多类活动。

潮u没有沉重的历史包袱，这会对未来的枝江产生什么影响呢...

## 六、结论总结

综合当前结果，可以给出总结：

> 乃琳鸣潮在当前观测窗口内构成了明显的增量入口；这批增量用户在后续流动上首先强烈流向嘉然，贝拉也有承接但显著较弱，同时乃琳自身的后续承接依然保持较高水平。
>
> 整体上，这批用户并不是简单地“从乃琳流失”，而是在乃琳打开入口之后，逐步演化为以嘉然为主要扩散方向、并部分形成多成员观看的团内扩张结构。
